# 对象存储工具包

使用对象存储服务所需的工具包，对象存储是一种专为海量非结构化数据设计的云存储架构，为客户提供海量、安全、高可靠、低成本的数据存储能力。

## 🚀 安装方法

### 方法一（推荐）

1. 登录 WordPress 管理员后台，进入 `Kratos` > `插件中心` 页面；

2. 找到 `对象存储工具包` 插件，单击 `立即安装` 按钮；

3. 安装完成后，将会看到「安装完成」的提示，然后单击 `启用插件` 按钮。

### 方法二

1. 下载最新的安装包文件；

2. 登录 WordPress 管理员后台，进入 `插件` > `添加插件` 页面，然后单击 `上传插件` 按钮；

3. 单击 `选择文件`, 选择刚刚下载的安装包文件，然后单击 `立即安装` 按钮；

4. 安装完成后，将会看到「插件安装成功」的提示，然后单击 `启用插件` 按钮。
